# Extracteur de Métadonnées d'Images (EXIF & XMP)

## Description
Ce logiciel Java permet d’extraire et d’analyser les métadonnées EXIF et XMP des images.  
Il offre une interface simple pour visualiser les informations techniques et faciliter la gestion et le tri des fichiers image.

## Fonctionnalités
- Extraction des métadonnées EXIF et XMP via les tags
- Visualisation des informations techniques des images
- Gestion et tri automatisé des fichiers image
- Interface utilisateur intuitive pour un accès rapide aux données

## Technologies utilisées
- Java (programmation orientée objet)
- Bibliothèques Java pour lecture des métadonnées
- Swing pour l’interface graphique
